# ☕ Coffee Dataset EDA

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a coffee sales dataset using Python.  
The analysis focuses on identifying sales trends, customer behavior, product performance, and business insights through data visualization and statistical analysis.

---

## 🎯 Objective
- Analyze coffee sales data
- Identify top-performing products
- Understand customer purchasing behavior
- Discover revenue trends and business insights
- Practice real-world data analysis using Python

---

## 📂 Dataset Information
The dataset includes:
- Coffee product categories
- Sales transactions
- Customer information
- Revenue and quantity details
- Product and order data

---

## 🛠️ Tools & Technologies Used
- Python

- Matplotlib
- Seaborn
- Jupyter Notebook
